# instalação (terminal bash)

Faça o clone do front-end 
```bash
git clone git@github.com:P4ul0Silva/planetFoneChat.git
```
Vá para
```bash
cd planetFoneChat
```
rode
```bash
yarn 
``` 
ou 
```bash
npm 
```

## Após a instalação 
rode
```bash
yarn dev
```
A aplicação vai estar rodando localmente
```bash
http://localhost:5173 
```
